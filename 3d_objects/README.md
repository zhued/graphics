## Key bindings:

- UP ARROW: Rotate X axis +
- DOWN ARROW: Rotate X axis -
- LEFT ARROW: Rotate Y axis +
- RIGHT ARROW: Rotate Y axis -
- m: Toggle between perspective and orthogonal
- +/-: Changes field of view for perspective
- a: Toggle axes
- w/s: Zoom in and out
- ESC: Exit
